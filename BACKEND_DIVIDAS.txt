[PERSISTÊNCIA DE LOGIN]
  - Frontend já usa localStorage para token e cache de usuário.
  - Sessão persiste até: (a) logout, (b) token expirado (401), (c) limpar dados do site.
  - Para sessões longas: backend deve usar JWT com expiração alta (ex.: 7–30 dias)
    ou implementar refresh token (POST /auth/refresh com refresh_token).
